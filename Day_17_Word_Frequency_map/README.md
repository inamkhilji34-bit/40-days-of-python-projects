# Day 17: Word Frequency Map

## Description
A Python program that analyzes text and counts how often each word appears.
It filters out common words (stop words) and displays the top 5 most frequent words.

## What It Does
- Takes a block of text as input
- Splits the text into individual words
- Ignores common/uninteresting words like "the", "a", "is"
- Counts how many times each word appears
- Displays the top 5 most frequent words

## How To Run
1. Make sure you have Python installed (Python 3.x recommended)
2. Clone or download this repository
3. Run the script:
   python word_frequency.py

## Example Output
Top 5 most frequent words:
1. cat      → 4 times
2. happy    → 3 times
3. sat      → 2 times
4. mat      → 2 times
5. purple   → 1 time

## Concepts Used
- Dictionaries
- Loops and conditionals
- String methods (.lower(), .split())
- Sorting with lambda functions
- Sets (for stop words)
