# 🐍 40 Days of Python

A daily Python coding challenge — one real project every day for 40 days, starting from fundamentals and working up through data structures, file I/O, APIs, linear algebra with NumPy, and full Object-Oriented Programming.

> **Goal:** Build a strong Python foundation, practice linear algebra in code, and create a GitHub profile worth showing.

---

## 👤 About Me

I'm learning Python and Linear Algebra with the goal of building toward Data Science and Machine Learning. This repo documents every project I build along the way — bugs, fixes, and all.

## 🚀 How to Run Any Project

```bash
# Clone the repo
git clone https://github.com/inamkhilji34-bit/40-days-of-python
cd 40-days-of-python

# Run any day
cd day-01-number-guesser
python main.py
```

> **Requires:** Python 3.10+ (for match/case support)
> For NumPy projects: `pip install numpy`
> For Matplotlib projects: `pip install matplotlib`

## 📌 Key Concepts Practiced

- `while True` + `try/except` + `break` — the standard Python input validation pattern
- `match/case` — Python 3.10+ alternative to `if/elif/else` chains
- `@property` decorator — controlled access to class attributes
- `np.linalg.solve()` — solving systems of linear equations
- Power iteration — approximating dominant eigenvalues without a library
- `json.load()` / `json.dump()` — persistent data storage without a database
