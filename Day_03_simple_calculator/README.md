# 🧮 Simple Calculator - Day 3 Project

A beginner-friendly command-line calculator built with Python that performs basic arithmetic operations.

## 📋 Project Description

This project is part of a Python learning journey (Day 3). It's a simple yet functional calculator that allows users to perform addition, subtraction, multiplication, and division operations through an interactive menu system.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with zero-division error handling)
- 🔄 Interactive menu system
- ✅ Input validation
- 🛡️ Error handling for invalid inputs

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your computer

### Installation

1. Clone or download this repository
2. Navigate to the project directory
3. Run the calculator:

python calculator.py


## 💻 Usage Example

=== Simple Calculator ===
1. Add
2. Subtract
3. Multiply
4. Divide
5. Exit

Choose an operation (1-5): 1
Enter first number: 10
Enter second number: 5
Result: 10 + 5 = 15

## 🐛 Error Handling

The calculator handles:
- **Division by zero**: Displays a friendly error message
- **Invalid menu choices**: Prompts user to select a valid option
- **Non-numeric inputs**: Catches and handles ValueError exceptions
